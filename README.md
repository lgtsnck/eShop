# eShop — Интернет-магазин на Django 6.0

Курсовой проект интернет-магазина, реализованный на стеке Python 3.13 и Django 6.0. Проект включает в себя каталог товаров (книги и канцтовары), корзину и систему управления заказами.

## ✅ Функционал
*   Разделение товаров по категориям (Книги, Канцтовары) с уникальными атрибутами.
*   Кастомизированная админ-панель с валидацией изображений и фильтрацией категорий.
*   Система корзины покупок и управления пользователями.
*   Использование современных возможностей Python 3.13 и Django 6.0.

## 🛠 Технологический стек
*   **IDE** Pycharm 2025.3.1.1+
*   **База данных** SQLite
*   **Python:** 3.13+
*   **Django:** 6.0+
*   **Pillow:** для обработки изображений.

## 🚀 Инструкция по запуску
### 1. Клонирование репозитория
```bash
git clone https://github.com/lgtsnck/eShop.git
cd eShop
```
### 2. Создание виртуального окружения (лучше автоматически ч-з IDE)
```bash
python -m venv venv
```
#### Активация виртуального окружения
```bash
# Активация для Windows:
venv\Scripts\activate

# Активация для Linux/macOS:
source venv/bin/activate
```
#### Переход в исходный каталог проекта
```bash
cd bookshop
```
### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```
### 4. Применение миграций (если хотите использовать свою БД, например, также SQLite по умолчанию, в проекте уже есть БД с примерами)
```bash
python manage.py makemigrations
```
```bash
python manage.py migrate
```
### 5. Создание суперпользователя (в случае выполнения пункта 4)
```bash
python manage.py createsuperuser
```
### 6. Данные для входа (Суперпользователя)
#### Если вы используете базу данных из репозитория, используйте следующие данные для входа в админку:
Логин: 
```bash
admin
```
Пароль: 
```bash
f123abc890xyz
```
### 6. Запуск сервера
```bash
python manage.py runserver
```
